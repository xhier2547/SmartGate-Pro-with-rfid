LED + Buzzer — 2-Way IoT Feedback

ESP32 รับคำสั่งจาก Server ผ่าน MQTT · แสดงผลทันทีที่ประตู

🟢 ผ่าน AI (ปกติ) LED เขียว 1 วิ · Beep 1800Hz 100ms 'pass'
🟡 มาสาย LED เหลือง 1.5 วิ · Beep 2 ครั้ง 'late'
🔴 AI ไม่ผ่าน LED แดง 2 วิ · Buzz ยาว 500Hz 1.5 วิ 'fail'
❓ บัตรไม่รู้จัก LED แดงกะพริบ 3× · Beep 3 ครั้งสั้น 'unknown'
🚪 เช็คเอาท์ (ออก) LED เขียวสั้น 0.6 วิ · Beep เบา 1600Hz 'checkout'
